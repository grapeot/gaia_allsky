为什么这件事有意思

这个实验解释城市为什么看不到银河。

光污染铺了一层底光

城市灯光给天空铺上了一层底光。星光本身的亮度不变。底光越强，能分辨的暗结构就越少。同一片天空始终在天上，变化的是我们能看到多少。

人眼极限随天空亮度而变

裸眼极限星等（NELM）说的是肉眼在最暗处还能看到多暗的星，这个数值由天空背景亮度和人眼灵敏度共同决定。Bortle 1 顶级暗空里，健康眼睛可以看到约 7.6–8.0 等；常说的肉眼可见 6 等，更接近普通暗空的实际情况。

星座只是一个看的方向

北斗七星从地球看过去连成一个勺子的形状，只是因为它们大致在我们视线的同一侧。如果移动到几十 pc 之外（1 pc 约 3.26 光年），近处的星在视野里会明显移位，勺子就散开了。银河带则来自更远处、更大范围的恒星分布，换一个角度看也不会消散。

交互结果 02

移动位置，星座散开，银河仍在。

前向透视：先看北斗，再转向银盘

轨迹分两段：第一段朝北斗移动，细线标出了七颗星在空间中位移的过程；第二段转向飞向银心方向上方，镜头始终对准银盘。这是一个普通的前向视角，就像相机取景一样只看前方区域。看的时候注意细线怎么从勺子的形状慢慢散开。预览视频无声，看画面变化即可。

VR 全天：同一条轨迹的球面视频

equirectangular 是一种把整个天球展开成 2:1 矩形图片的格式，放进手机或 VR 头显就可以转头看任意方向。它和前向视频走的是同一条空间轨迹，但保留了你周围整个天空的信息。画面边缘的拉伸变形是球面展开到平面的几何效果，属于格式特征。预览视频无声，看画面变化即可。

项目的边界

这个项目的目标是定性正确和可解释性。测光级精度验证不在设计范围之内。

必须说清楚的部分

Gaia 视差数据覆盖的是太阳附近的数据球。

飞行视频诚实地展示星座散开、星场重投影、以及数据覆盖的物理边界。当你在视频中移动到数据球边缘时，会看到恒星开始变稀疏：这就是可观测范围的尽头。人类目前还没有银河系的真实俯视照片；常见的银河俯视图来自多波段观测和模型反演。

页面图像还经过一道 tone mapping（亮度映射）。恒星的星等转物理亮度、天球坐标投影、视差重投影：这些步骤有物理公式作为锚点。最终要把天文级动态范围的亮度放进普通屏幕（SDR），就需要额外的亮度压缩。代码把这两层分开处理，避免将显示参数说成物理参数。

复现与二次开发

代码、数据脚本、测试和渲染命令都在 GitHub 仓库里。

正常渲染和测试不需要密钥。Gaia 缓存和完整输出留在本地，Git 里只放代码、文档、测试和网页压缩资产。

入口命令：uv venv、uv pip install -r requirements.txt、python -m pytest tests/ -q。
