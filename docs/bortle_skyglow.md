# Bortle 暗空等级 → 天空辉光量化映射

光污染 knob 的物理依据。天空辉光（skyglow）是叠加在星空上的均匀加性背景光，
Bortle 1-9 对应不同的天空背景面亮度（mag/arcsec²）。

## 主对照表

| Bortle | 天空面亮度 mag/arcsec² | 肉眼极限星等 NELM | 描述 |
|---|---|---|---|
| 1 | 21.99–22.0 | 7.6–8.0 | 极致暗空，荒漠/高山；黄道光、银河投影清晰 |
| 2 | 21.89–21.99 | 7.1–7.5 | 真正暗空场地 |
| 3 | 21.69–21.89 | 6.6–7.0 | 乡村星空，地平线附近轻微光污染 |
| 4 | 20.5–21.69 | 6.1–6.5 | 乡村/郊区过渡，远处城市光穹明显 |
| 5 | 19.5–20.5 | 5.6–6.0 | 郊区，银河微弱、近地平线消失 |
| 6 | 18.9–19.5 | 5.1–5.5 | 明亮郊区，天顶泛灰白，银河几乎不可见 |
| 7 | 18.4–18.9 | 4.6–5.0 | 城郊过渡，整片天空泛灰白 |
| 8 | ~18.0 | 4.1–4.5 | 城市，可读报纸，只见亮星行星 |
| 9 | ~17.5 | ≤4.0 | 市中心，仅月亮行星和最亮恒星 |

渲染用中值（`BORTLE_MU`）：22.0 / 21.9 / 21.8 / 21.1 / 20.0 / 19.2 / 18.6 / 18.0 / 17.5。

## 物理

- **辉光是 additive 的**：城市灯光经大气分子和气溶胶散射回视线，叠加自然夜天光
  （气辉、黄道光、散射星光）。线性光强空间里：观测总亮度 = 天体亮度 + 辉光基底。
- **mag/arcsec² 是对数标度**：数值越小=天空越亮=污染越重。L ∝ 10^(−0.4·μ)。
  Bortle 9 (17.5) 比 Bortle 1 (22.0) 数值小 4.5 等，背景亮约 60 倍。
- 渲染里先把 μ 转线性，作为均匀基底叠加到星空线性亮度，再做 tonemap，
  "压低对比、淹没暗星"才物理自洽。

## 标定

`SKYGLOW_SCALE`：μ（每角秒²）与渲染像素尺度差多个量级，标定一个缩放把物理梯度
映到视觉量级。锚点 Bortle5（μ=20，郊区）辉光 ≈ 银河带典型线性亮度，其余按 μ 自动拉开。
