# Gaia 全天星图渲染

用真实恒星星表（Gaia DR3，亮度 cutoff 子集）渲染一张**全天投影图**，看银河（密集恒星带）能否从
逐颗真实恒星自然涌现。

## 由来

源自 turquoise_band（月食模拟）项目里做的一个物理正确的星场渲染管线：
- **星等 → 亮度**：`L = 10^(-0.4·(m − m_ref))`，物理正确的相对亮度（tone mapping）。
- **B-V 色指数 → 星色**：恒星色温（蓝白星 vs 橙红星），真实星色。

那个管线在一小片天区（金牛座）上验证过。本项目把它推到**全天 Gaia 星表**——如果管线对，
银河（恒星密度最高的带）应该自然涌现，不需要任何"画银河"的特殊处理。这是第一性原理的检验。

## Scope

- **数据**：Gaia DR3，按亮度 cutoff（如 G<8–10）取子集（全表 20 亿颗太大；cutoff 子集几十万到几百万颗，
  够看银河结构且数据量可控）。
- **投影**：全天投影（360°×180°），equirectangular 或 Mollweide。
- **渲染**：复用星等→亮度 + B-V→星色管线。
- **执行顺序**：(1) 低分辨率 SDR 验证银河涌现 → (2) 高分辨率 SDR → (3) HDR。

## 结果

银河从逐颗真实恒星自然涌现，零特殊处理，验证了第一性原理管线正确：

- **低分辨率 SDR**（G<8，6.3 万颗）：银道面/高银纬恒星密度比 **10.4×**，银河带可辨。
- **高分辨率 SDR**（G<11，124 万颗）：`outputs/gaia_g11_mollweide.png`。银河带、银心方向的
  暗尘埃裂缝（Great Rift）清晰；密度比降到 2.9×（暗星更均匀分布，但累加亮度银河带仍浓密）。
- **HDR**（16bit 线性 TIFF）：`outputs/gaia_g11_mollweide_hdr.tif`。峰值/中位动态范围 **21510:1
  （≈14.4 stops）**——SDR 8bit 只有 256:1（8 stops），必须在"银心过曝"和"暗星淹没"间二选一；
  HDR 同时保留两端，后期可分别 grade。`gaia_g11_hdr_logpreview.png` 是对数压缩预览。
